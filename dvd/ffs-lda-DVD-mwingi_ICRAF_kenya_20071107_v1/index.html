PROJECT CD – Mwingi district

This project CD is the result of a study on land degradation assessment for the Farmer Field School Approaches to Overcome Land Degradation in Agro-Pastoral Areas of Kenya, made in 2007. The study was done at different scales. For the whole of Kenya vegetation data with 8 km resolution going back to 1981 were used to identify areas where vegetation changes (decreases) have been anomalously large. For the districts of Mbeere, Mwingi and Narok, vegetation data with 250 m resolution going back to 2000 were combined with rainfall data for the same period to identify regions with vegetation changes unaccounted for by changes in rainfall. A field study was undertaken in June 2007 to collecting ground data, and document the landscape using panorama photographs (that covers the whole horizon).

The project CD (version 1.0) contains the assessment report; the data used for the land degradation baseline survey; DIVA-GIS a free GIS (Geographic Information System) software; and a training module for doing land degradation assessment using DIVA-GIS and time series of vegetation data. Descriptions of the included datasets are linked in the web catalog of each data set (see table below). The project CD also contains ArcView GIS project files with all the framework data included. The ArcView project files are in two versions, one including all the framework datasets (demands that the user has the extension Spatial Analyst installed), and one version only containing vector files (can be used with an ordinary ArcView GIS license).The table below shows the folders (files) of the documents and other resources, and if viewed in a web-browser, the links can be clicked to open up the different resources.
