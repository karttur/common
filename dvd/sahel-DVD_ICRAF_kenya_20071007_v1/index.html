West African Drylands Project

An Ecosystem Approach to Restoring West African Drylands and Improving Livelihoods through Agroforestry-based Land Management Interventions.

A United Nations Environment Programme (UNEP) project conducted in partnership with the World Agroforestry Centre (ICRAF), the Centre for Environmental Policy of the University of Florida, and the Governments of Burkina Faso, Mali, Mauritania, Niger and Senegal. The project is funded by the Government of Norway.

Project CD created by Thomas Gumbricht, www.mapjourney.com

PROJECT CD

This project CD is the result of a study on land degradation assessment for the West African Drylands Project.

The project CD (beta version 1.0) contains the data used for the land degradation baseline survey, a free GIS (Geographic Information System) software, and a training module for doing land degradation assessment using GIS and time series of vegetation data. All documentation is available as pdf files. The table below shows the folders (files) of the documents and other resources, and if viewed in a web-browser, the links can be clicked to open up the different web-pages.
